tools/ — optional local helpers
================================

Purpose
-------
Placeholder for optional helper scripts you add on your own PC (installers,
one-off fixes, personal automation). Nothing here is required to run DictaDesk.

Repo-maintained dev scripts
---------------------------
See scripts/ — e.g. export_prompts.py to refresh prompts/*.txt from code.

Git / GitHub
------------
This folder is tracked via this README only (empty dirs are not pushed).
Add your own files locally; they stay untracked unless you commit them on purpose.
