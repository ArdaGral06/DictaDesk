memory/ — DictaDesk runtime folder
==================================

Purpose
-------
Agent memory: preferences, notes, routines in long_term.json (created from
long_term.json.example on first run).

Auto-created
------------
Folder: yes (start.bat / install.ps1 / ensure_dirs).
long_term.json: install.ps1 or first memory load if missing.

Git / GitHub
------------
long_term.json is gitignored (personal data). Only this README and
long_term.json.example are tracked in the repo.

To reset: delete long_term.json and restart.
