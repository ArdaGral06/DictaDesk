memory/ — DictaDesk runtime folder
==================================

Purpose / Amaç
--------------
Agent memory: preferences, notes, routines in long_term.json (created from .example on first run).
Ajan hafızası: tercihler, notlar, rutinler — long_term.json (ilk çalıştırmada .example'dan kopyalanır).

Auto-created / Otomatik oluşturulur
-----------------------------------
Folder: yes (start.bat / install.ps1 / ensure_dirs).
long_term.json: install.ps1 or first memory load if missing.
Klasör: evet. long_term.json: install.ps1 veya ilk yüklemede.

Git / GitHub
------------
long_term.json is gitignored (personal data). Only this README and long_term.json.example are in the repo.
long_term.json GitHub'a gitmez. Repoda yalnızca bu README ve long_term.json.example vardır.

To reset / Sıfırlamak için: delete long_term.json and restart.
