================================================================================
  DictaDesk - Getting Started (for everyone)
================================================================================

DictaDesk lets you control your Windows PC with your voice or keyboard.


WHERE TO DOWNLOAD DICTADESK
----------------------------

  GitHub: https://github.com/ArdaGral06/DictaDesk

  Easiest: open the link -> Code -> Download ZIP -> extract the folder.
  Or: git clone https://github.com/ArdaGral06/DictaDesk.git

  All setup files (install.bat, start.bat) are inside that folder.


WHAT YOU NEED BEFORE STARTING
-----------------------------

  1. Windows 10 or 11 (64-bit)
  2. Internet connection (for first-time setup downloads)
  3. Python 3.10 or newer - install ONCE from https://www.python.org/downloads/
     IMPORTANT: During Python setup, check "Add Python to PATH" at the bottom.


FOUR STEPS TO RUN DICTADESK
---------------------------

  STEP 0 - Download DictaDesk from GitHub (see link above), extract ZIP

  STEP 1 - Install Python (one time only)
          Download from python.org, run installer, check "Add Python to PATH".

  STEP 2 - Run install.bat (one time only)
          Double-click install.bat in this folder.
          Wait until you see "Setup complete". This can take 5-20 minutes.
          The installer does everything else for you automatically.

  STEP 3 - Run start.bat (every time you use DictaDesk)
          Double-click start.bat.
          If you skipped Step 2, start.bat will run the installer for you first.

  OPTIONAL - Tesseract OCR (only for "click text on screen" features)
          Install from https://github.com/UB-Mannheim/tesseract/wiki
          Then run install.bat again once for Turkish language pack.
          DictaDesk works WITHOUT Tesseract for most commands.


THREE STEPS TO RUN DICTADESK (short version)
--------------------------------------------

  Download from GitHub -> install.bat once -> start.bat every time
  (Plus Python from python.org before install.bat)


WHAT EACH FILE DOES
-------------------

  install.bat
      Opens a setup window and runs install.ps1.
      You normally run this ONCE after downloading DictaDesk.

  install.ps1
      The real installer (runs automatically - do not double-click unless asked).
      It creates .venv, installs Python packages, downloads Piper voice engine,
      Playwright browser, and creates config files.

  start.bat
      Starts DictaDesk. Use this every day.
      Checks that setup finished; if not, runs install.bat first.

  voice_control.py
      The main program (start.bat runs this for you).


WHAT install.bat INSTALLS FOR YOU (automatic)
----------------------------------------------

  [x] Python virtual environment (.venv folder)
  [x] All required Python libraries (Whisper, Playwright, etc.)
  [x] Playwright Chromium browser (~150 MB) for web commands
  [x] Piper TTS program + English voice model (required to start the app)
      -> install.bat downloads piper.exe automatically. You NEVER run piper.exe yourself.
      -> Official sources: github.com/rhasspy/piper + huggingface.co/rhasspy/piper-voices
  [x] secrets.json and memory files from templates
  [x] Turkish OCR file (tur.traineddata) IF Tesseract is already installed


WHAT YOU MAY INSTALL MANUALLY (optional)
----------------------------------------

  Tesseract OCR - only if you want "click this text on screen" features
      Download: https://github.com/UB-Mannheim/tesseract/wiki
      After installing Tesseract, run install.bat again once.

  Groq API key - free, for smart natural-language commands (recommended)
      Get a key at: https://console.groq.com
      Enter it when DictaDesk asks on first run (saved in secrets.json).

  Local LLM (.gguf file) - only if you choose "Local Agent" at startup
      NOT required for normal use. Groq API is easier and free.


FIRST RUN - RECOMMENDED MENU CHOICES
------------------------------------

  When DictaDesk starts, it asks a few questions. For most users pick:

    UI language     : 1 (Turkish) or 2 (English)
    Speech (STT)    : 1 (Whisper) - first use downloads a model (~500 MB)
    Voice output    : 1 (Off) - Piper is installed but you can skip speaking
    LLM / brain     : 3 (Groq API) - paste free API key when asked
    Vision (VLM)    : 1 (Off) unless you need screenshot-based clicking

  Then from the main menu:
    3 = Self-check (verify everything works)
    1 = Control mode (press F9 to speak, or type commands)


EXAMPLE COMMANDS (after entering Control mode)
----------------------------------------------

  "sesi yuzde elli yap"     - set volume to 50%
  "discord kapat"           - close Discord
  "youtube da X ara"        - search YouTube (needs LLM + web automation)
  "bilgisayari kapat"       - shuts down PC (will ask for confirmation)


TROUBLESHOOTING
---------------

  "Python not found"
      Install Python from python.org and check "Add Python to PATH".
      Close all windows and run install.bat again.

  "Virtual environment not found"
      Run install.bat (or just use start.bat - it runs install for you).

  "Piper missing"
      Run install.bat again with internet connected.

  Setup script error with strange characters
      Re-download the latest DictaDesk ZIP from GitHub (encoding fix).

  Something else broken
      Open an issue: https://github.com/ArdaGral06/DictaDesk/issues


FOLDER QUICK REFERENCE
----------------------

  install.bat / start.bat     - use these
  recordings/                 - your voice recordings (can delete old files)
  secrets.json                - your API keys (never share this file)
  commands.json               - your custom voice commands
  .venv/                      - Python environment (do not delete)
