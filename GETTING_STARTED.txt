================================================================================
  DictaDesk - Getting Started (English)
================================================================================

DictaDesk lets you control your Windows PC with your voice or keyboard.

Turkish guide: open GETTING_STARTED_TR.txt in this folder.


WHERE TO DOWNLOAD
-----------------

  https://github.com/ArdaGral06/DictaDesk

  Easiest: Code -> Download ZIP -> extract.
  Or: git clone https://github.com/ArdaGral06/DictaDesk.git


WHAT YOU NEED
-------------

  1. Windows 10 or 11 (64-bit)
  2. Internet (for first-time setup downloads)

  That's it. install.bat installs Python and Tesseract for you if they
  are missing - you do NOT need to install anything by hand.


TWO STEPS (fully automatic)
---------------------------

  STEP 1 - Double-click install.bat (once)
          It installs EVERYTHING automatically:
            - Python 3.12 (if missing) + adds it to PATH
            - Tesseract OCR + Turkish pack + adds it to PATH
            - All Python packages, Piper TTS, Playwright browser
            - The correct GPU backend for your card (NVIDIA / AMD / CPU)
          A Windows security prompt may appear ONCE for Tesseract - click Yes.
          Wait for "Setup complete".

  STEP 2 - Double-click start.bat (every time you use DictaDesk)

  Note: no manual PATH editing. If you are on a locked-down PC where the
  auto-installer is blocked, see README.md "Adding Python and Tesseract to
  PATH" as a fallback only.


FIRST RUN - RECOMMENDED CHOICES
---------------------------------

  UI language     : Turkish (1) or English (2)
  Speech (STT)    : 1 Whisper or 3 Groq API
  Voice (TTS)     : 1 Off
  LLM / Agent     : 3 Groq API (paste free key when asked)
  Vision (VLM)    : 1 Off

  Main menu: 3 = Self-check   |   1 = Control mode (Ctrl+Shift+6)


EXAMPLE COMMANDS
----------------

  "sesi yuzde elli yap"     - set volume to 50%
  "discord kapat"           - close Discord
  "notepad ac" / "open notepad"


CHANGE API KEY LATER
--------------------

  Windows Credential Manager -> Windows Credentials -> DictaDesk entries
  Edit the password field (JSON with "api_key"). Restart DictaDesk.

  Or delete the entry and restart; the app will ask again.


CHANGE API PROVIDER / MODEL
---------------------------

  Edit providers.json, llm_providers.json, vlm_providers.json, or
  tts_providers.json — see API_PROVIDERS.md. Restart after saving.


TROUBLESHOOTING
---------------

  Python auto-install failed -> Rare. Install Python 3.12 from python.org
                                with "Add to PATH", then re-run install.bat
  Tesseract not installed    -> Re-run install.bat with internet on
                                (click Yes on the security prompt)
  Virtual env missing        -> Run install.bat or start.bat
  Piper missing              -> Run install.bat again
  Other issues               -> https://github.com/ArdaGral06/DictaDesk/issues


USEFUL FOLDERS
--------------

  install.bat / start.bat   - setup and launch
  recordings/               - voice recordings (safe to delete old files)
  secrets.json              - never share (API metadata)
  commands.json             - your custom commands
