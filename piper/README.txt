Piper executable folder
=======================

AUTOMATIC (normal setup)
------------------------
install.bat / install.ps1 download piper.exe here for you.
You do NOT need to run piper.exe yourself — DictaDesk calls it internally.

Expected file after setup:
  piper/piper.exe

Voice models (auto-downloaded to):
  tts_models/piper/en_US-joe-medium.onnx
  tts_models/piper/en_US-joe-medium.onnx.json


MANUAL (only if install.bat failed)
-----------------------------------
Download piper.exe from the official release:
  https://github.com/rhasspy/piper/releases

Place the file here (do not double-click to run it):
  piper/piper.exe

Then re-run install.bat or download voice files to tts_models/piper/
