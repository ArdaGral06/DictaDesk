logs/ — DictaDesk runtime folder
=================================

Purpose / Amaç
--------------
Rotating log file dictadesk.log (warnings/errors and optional debug detail).
Dönen log dosyası dictadesk.log (uyarılar, hatalar ve isteğe bağlı ayrıntılar).

Auto-created / Otomatik oluşturulur
-----------------------------------
Yes — when logging starts (first menu load, Self-check, or any run that calls setup_logging).
Evet — logging başladığında (ilk menü, Self-check veya setup_logging çağrıldığında).

Git / GitHub
------------
Everything here except this README.txt is gitignored (*.log files).
Bu klasördeki log dosyaları (README hariç) GitHub'a gitmez.
