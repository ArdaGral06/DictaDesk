recordings/ — DictaDesk runtime folder
======================================

Purpose / Amaç
--------------
Microphone captures (.wav) saved during control mode.
Kontrol modunda mikrofondan kaydedilen ses dosyaları (.wav) buraya yazılır.

Auto-created / Otomatik oluşturulur
-----------------------------------
Yes — on first run of start.bat / main.py, install.ps1, or Self-check (option 3).
Evet — start.bat, install.ps1 veya Self-check (3) ile ilk çalıştırmada oluşur.

Git / GitHub
------------
Everything here except this README.txt is gitignored. Do not commit local recordings.
Bu klasördeki dosyalar (README hariç) GitHub'a gitmez; yerel kayıtları push etmeyin.
